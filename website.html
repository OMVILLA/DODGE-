Windows & macOS Shortcut Keys Guide

Shortcut Keys Guide for Windows & macOS

Windows Shortcut Keys

General

  • Copy
    CtrlC
  • Cut
    CtrlX
  • Paste
    CtrlV
  • Undo
    CtrlZ
  • Redo
    CtrlY
  • Select All
    CtrlA
  • Find
    CtrlF

System & Navigation

  • Open Task Manager
    CtrlShiftEsc
  • Switch between open apps
    AltTab
  • Lock your PC
    WindowsL
  • Open Start Menu
    Windows
  • Open Settings
    WindowsI
  • Show Desktop
    WindowsD
  • Snip & Sketch (Screenshot tool)
    WindowsShiftS

Text Editing

  • Move cursor to beginning of line
    Home
  • Move cursor to end of line
    End
  • Delete whole word
    CtrlBackspace
  • Select word
    CtrlShift← / →

macOS Shortcut Keys

General

  • Copy
    ⌘C
  • Cut
    ⌘X
  • Paste
    ⌘V
  • Undo
    ⌘Z
  • Redo
    ⌘⇧Z
  • Select All
    ⌘A
  • Find
    ⌘F

System & Navigation

  • Force Quit Applications
    ⌘⌥Esc
  • Switch between open apps
    ⌘Tab
  • Lock Screen
    ^⌘Q
  • Open Spotlight Search
    ⌘Space
  • Show Desktop
    fnF11
  • Screenshot selection
    ⌘⇧4

Text Editing

  • Move cursor to beginning of line
    ⌘←
  • Move cursor to end of line
    ⌘→
  • Delete whole word
    ⌥⌫
  • Select word
    ⇧⌥← / →
